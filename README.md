# Smart Billing System

A responsive web-based Smart Billing System built using HTML, CSS, and JavaScript.
## 🚀 Live Demo

👉 **[Open Smart Billing System](https://digitalbilling.netlify.app/)**

## ✨ Features

- Secure Login
- Dashboard
- Billing with GST Calculation
- Inventory Management
- Customer Management
- Sales Reports
- Printable Bills
- Responsive Design
- LocalStorage Data Storage

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- LocalStorage
- Netlify

## 📦 Installation

1. Clone the repository
2. Open `index.html` in your browser
